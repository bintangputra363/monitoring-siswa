Introduction

Selamat datang di Monitor Siswa, aplikasi manajemen siswa berbasis web.

Monitor Siswa adalah aplikasi yang dirancang untuk membantu sekolah dalam mengelola data siswa, guru, kelas, dan kegiatan. Aplikasi ini dibangun menggunakan PHP 8.2 dan Bootstrap v5.3.3, dengan antarmuka yang responsif dan mendukung berbagai perangkat.

Fitur utama aplikasi ini meliputi:

  • Manajemen Guru: Tambahkan dan kelola data guru dengan mudah.
  • Manajemen Siswa: Tambahkan dan kelola data siswa secara efisien.
  • Mapping Guru ke Kelas: Hubungkan guru dengan kelas yang mereka ajar.
  • Mapping Siswa ke Kegiatan: Hubungkan siswa dengan kegiatan tertentu.
  • Monitoring: Pantau progress siswa dalam berbagai kegiatan.

Aplikasi ini mendukung mode terang dan gelap, serta dapat disesuaikan dengan kebutuhan sekolah Anda.

Panduan Penggunaan Aplikasi

Setelah selesai melakukan setup, berikut adalah langkah-langkah untuk menggunakan aplikasi:

  1. Login: Masuk ke aplikasi menggunakan akun admin yang telah dibuat.
  2. Tambah Guru: Gunakan menu Register Guru di bagian atas untuk menambahkan guru baru.
  3. Tambah Siswa: Gunakan menu Tambah Siswa untuk menambahkan data siswa.
  4. Mapping Guru ke Kelas: Gunakan fitur Mapping Guru untuk menghubungkan guru dengan kelas tertentu.
  5. Mapping Siswa ke Kegiatan: Gunakan fitur Mapping Siswa untuk menghubungkan siswa dengan kegiatan tertentu.
  6. Monitoring: Gunakan halaman Dashboard untuk memantau progress siswa dan kegiatan.

Menjalankan Aplikasi dari Folder Starterkit

                                

                                    Untuk menjalankan aplikasi ini, pastikan Anda mengikuti langkah-langkah berikut:
                                

                                
    
                                    
  1. 
                                        Pastikan Anda telah menginstal XAMPP atau server lokal lainnya di komputer Anda.
                                    
    2. 
                                    
  2. 
                                        Ekstrak file aplikasi ke dalam folder htdocs di direktori instalasi XAMPP Anda. Contoh:
                                        
    C:/xampp/htdocs/monitor-siswa/Starterkit
    
                                    
    3. 
                                    
  3. 
                                        Buka file config.php di folder Starterkit/partials dan sesuaikan kredensial database Anda:
                                        
    define('DB_SERVER', 'localhost');
define('DB_USERNAME', 'root');
define('DB_PASSWORD', '');
define('DB_NAME', 'monitor_siswa');
                                        
    
                                    
    4. 
                                    
  4. 
                                        Jalankan XAMPP dan aktifkan modul Apache dan MySQL.
                                    
    5. 
                                    
  5. 
                                        Buka browser Anda dan akses aplikasi melalui URL berikut:
                                        
    http://localhost/monitor-siswa/Starterkit/auth-login.php
    
                                    
    6. 
                                    
  6. 
                                        Login menggunakan akun admin yang telah dibuat, atau buat akun baru jika diperlukan.
                                    
    7. 
                                

                                

                                    Setelah langkah-langkah di atas selesai, Anda dapat mulai menggunakan aplikasi untuk mengelola data siswa, guru, kelas, dan kegiatan.
